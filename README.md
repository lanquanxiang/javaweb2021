2021级软件工程、网络工程课程教学代码，Web前端开发技术课堂笔记。

#### 开发软件
1. 开发工具：Eclipse 2023
2. Java版本：JDK 11
3. 服务器容器： Tomcat 9.0
4. 数据库： MySQL 8.0
5. 数据库可视化工具：DBeaver 或 SQLYog
