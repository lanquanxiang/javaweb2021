# 1. 预编译的命令对象
## 1. Statement存在的问题
1. 编写SQL不方便，需要拼接参数
2. 存在安全问题，SQL注入式攻击（在参数中加入特殊字符绕过SQL判断）eg. 1'OR'1'='1
	解决办法：1. 不允许输入特殊字符；2. 使用预编译的命令对象
## 2. PreparedStatement的优点
1. 是Statement子接口
2. 优点：1、易于编写；2、易于维护；3、可读性更高；4、安全性更高（防止SQL注入）；5、执行效率更高（一次编译）
